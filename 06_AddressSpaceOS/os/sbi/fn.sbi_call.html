执行 SBI 调用

这是所有 SBI 接口的底层实现，通过 ecall 指令从监督者模式 陷入机器模式，调用 SBI 固件提供的服务。

§Arguments

  • eid - 扩展 ID，标识要调用的 SBI 扩展
  • fid - 函数 ID，标识扩展内的具体函数
  • arg0 - 第一个参数
  • arg1 - 第二个参数
  • arg2 - 第三个参数

§Returns

返回 SbiRet 结构，包含错误码和返回值

§Safety

使用内联汇编执行 ecall 指令，假设 SBI 固件正确实现。

§寄存器约定

  • a7 (x17): 扩展 ID
  • a6 (x16): 函数 ID
  • a0-a2 (x10-x12): 函数参数
  • 返回：a0 为错误码，a1 为返回值
