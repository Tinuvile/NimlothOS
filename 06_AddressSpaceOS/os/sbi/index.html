§SBI (Supervisor Binary Interface) 调用模块

提供与 RISC-V SBI 固件的交互接口，实现操作系统内核与底层固件之间的通信。 SBI 是 RISC-V 架构中监督者模式软件与机器模式固件之间的标准接口。

§支持的 SBI 扩展

§Legacy 扩展 (0.1 版本)

  • 控制台输入输出
  • 时钟设置
  • IPI (处理器间中断)
  • TLB 管理
  • 系统关闭

§系统复位扩展 (SRST)

  • 系统关闭和重启功能
  • 支持不同的复位类型和原因

§调用约定

SBI 调用使用 ecall 指令从 S 模式陷入 M 模式：

  • a7 寄存器：扩展 ID (EID)
  • a6 寄存器：函数 ID (FID)
  • a0-a2 寄存器：函数参数
  • 返回值通过 a0 (错误码) 和 a1 (返回值) 寄存器传递
