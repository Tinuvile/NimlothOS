pub fn shutdown(failure: bool) -> !
Expand description

关闭（或复位）系统

使用 SBI System Reset 扩展请求系统关闭。根据 failure 参数选择 关闭原因码，以便固件或上层记录关机原因。

§Arguments

  • failure - 是否因为系统故障而关闭
    • false：NoReason
    • true：SystemFailure

§Behavior

  • 正常情况下，此函数不会返回。
