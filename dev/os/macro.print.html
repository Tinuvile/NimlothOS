格式化打印宏（不换行）

类似于标准库的 print! 宏，将格式化的文本输出到控制台， 不会在末尾添加换行符。

§语法

print!("format string", arg1, arg2, ...);

§格式化支持

支持 Rust 标准的格式化语法：

  • {} - 默认格式化
  • {:?} - Debug 格式化
  • {:x} - 十六进制格式化
  • {:>10} - 右对齐，宽度为10

§Examples

print!("Hello, ");
print!("world!");        // 输出: Hello, world!
print!("Value: {}", 42); // 输出: Value: 42
