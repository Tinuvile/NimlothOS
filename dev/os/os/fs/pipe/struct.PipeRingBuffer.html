Expand description

管道环形缓冲区

使用固定大小的数组作为底层存储，head 指向下一个可读位置，tail 指向下一个可写位置。 通过 status 区分满/空/正常，避免歧义。write_end 记录写端是否仍存活，用于 EOF 判断。
