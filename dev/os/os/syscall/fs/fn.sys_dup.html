pub fn sys_dup(fd: usize) -> isize
Expand description

系统调用：复制文件描述符（dup）

实现 dup(2) 的核心语义：为已打开的文件描述符分配一个新的最小可用 文件描述符编号，并让两者引用同一个底层文件对象（共享偏移与状态）。

§Arguments

  • fd - 需要复制的已有文件描述符

§Returns

  • 成功：返回新的文件描述符编号
  • 失败：返回 -1（如 fd 无效或未打开）

§共享语义

  • 新旧两个 fd 指向同一 File 对象，读写偏移共享
  • 关闭任意一个 fd 不影响另一个 fd 的有效性（引用计数减少）
