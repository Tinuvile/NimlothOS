pub fn get_time() -> usize
Expand description

获取系统时间（时钟周期）

读取 RISC-V 的 time CSR 寄存器，返回自系统启动以来经过的 时钟周期数。这是最精确的时间测量方式。

§Returns

返回时钟周期计数，单位为时钟周期（通常为纳秒级精度）

§Usage

let start = get_time();
// 执行某些操作
let end = get_time();
let cycles = end - start;  // 经过的时钟周期数

§Note

时钟周期数会在长时间运行后溢出，但对于大多数应用场景足够使用。
