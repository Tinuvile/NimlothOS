pub fn get_time_ms() -> usize
Expand description

获取系统时间（毫秒）

将时钟周期数转换为毫秒，提供更直观的时间测量单位。 转换基于系统时钟频率 CLOCK_FREQ 进行。

§Returns

返回自系统启动以来经过的毫秒数

§Precision

精度取决于时钟频率和整数除法的舍入误差，通常为毫秒级精度。

§Usage

let start_ms = get_time_ms();
// 执行某些操作
let end_ms = get_time_ms();
let elapsed_ms = end_ms - start_ms;  // 经过的毫秒数

§Implementation

通过除法 time / (CLOCK_FREQ / MSEC_PER_SEC) 将时钟周期转换为毫秒。 这等价于 (time * MSEC_PER_SEC) / CLOCK_FREQ，但避免了大数乘法的溢出风险。
