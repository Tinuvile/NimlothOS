Expand description

获取指定块的缓存

全局函数，通过全局缓存管理器获取指定块的缓存。 这是访问块缓存的主要接口。

§Arguments

  • block_id - 要获取缓存的块号
  • block_device - 块设备引用

§Returns

块缓存的引用计数智能指针

§Examples

let cache = block_cache(block_id, block_device);
cache.lock().modify(0, |data: &mut DataBlock| {
    data[0] = 0x42;
});
