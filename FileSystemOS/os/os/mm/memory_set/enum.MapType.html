Expand description

内存映射类型

定义虚拟页面到物理页帧的映射方式，影响地址转换的行为和性能。

§映射类型对比

类型地址关系用途优势劣势
IdenticalVA = PA内核段、设备简单、高效无隔离
FramedVA ≠ PA用户程序、堆灵活、安全复杂、开销

Variants§

§

Identical

恒等映射

虚拟地址直接等于物理地址，不需要额外的物理页帧分配。 主要用于内核代码段、数据段以及设备内存映射。

§特点
  • 地址转换开销最小
  • 地址可预测，便于调试
  • 适合需要直接访问物理地址的场景
§

Framed

帧映射

每个虚拟页面映射到独立分配的物理页帧，提供完整的 地址空间隔离和内存保护功能。

§特点
  • 支持地址空间隔离
  • 可以实现写时复制、懒加载等高级功能
  • 需要维护虚拟页到物理页的映射表
