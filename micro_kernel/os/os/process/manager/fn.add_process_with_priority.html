向全局就绪队列添加进程

  • 将 process 写入 PID2PCB 映射，便于通过 PID 查找 PCB
  • 将进程入队到全局 PROCESS_MANAGER 的就绪队列，等待调度

§参数

  • process - 待加入调度的进程控制块

§复杂度

  • O(1) 摊还时间；队列扩容时可能触发 O(n) 拷贝 向指定优先级队列添加进程

将进程添加到指定优先级的就绪队列中，同时更新 PID 映射。

§参数

  • process - 待加入调度的进程控制块
  • priority - 目标优先级队列（0为最高优先级）
